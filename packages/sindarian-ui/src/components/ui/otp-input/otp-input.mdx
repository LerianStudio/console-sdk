# InputOTP

A one-time password input component built on top of [input-otp](https://github.com/guilhermerodz/input-otp). Renders individual digit slots with automatic focus management, paste support, and browser autofill compatibility.

### Basic

A 6-digit OTP input with two groups separated by a divider.
